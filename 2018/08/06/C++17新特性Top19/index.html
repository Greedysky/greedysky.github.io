什么是 C++17？

       C++17（或 C++1z）是继 C++14 之后 C++ 编程语言 ISO/IEC 标准的下一次修订的非正式名称。C++17 现在功能已齐全，正在成为国际标准的路上。它的规范已于 2017 年 3 月达到国际标准草案的阶段。
